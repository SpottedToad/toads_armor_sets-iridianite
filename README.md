This is the mod for the in-progress 2a2c1n minecraft server. 
It uses a version of azurelib that is a fork of geckolib 4, which means that it runs in Java 21. 
Hopefully this doesnt cause major issues down the line.

The required mods, as of the last README.md update, are as follows. 
(does not include the mods required by main datapack):

(unused) Geckolib 3.1.40
Azurelib 3.2.10
Oh My, Meteors! 1.0.0+1.19.2
